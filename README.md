#  How It Works(Extension)

This repository is automatically updated using **LeetHub v2**.

#Step 1
Install the **LeetHub v2** Chrome Extension.

# Step 2
Sign in with your GitHub account.

# Step 3
Connect or create your LeetCode repository.

# Step 4
Log in to your LeetCode account.

# Step 5
Solve any LeetCode problem.

# Step 6
Click **Submit**.

# Step 7
If your solution is Accepted, LeetHub automatically pushes the solution to this GitHub repository.
No manual upload is required.
